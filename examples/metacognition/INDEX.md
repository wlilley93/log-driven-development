# Metacognition journal: index

> **What this is.** The one-line-per-beat index of the journal. It exists so a reader (human or agent) can scan
> the whole line of reasoning without opening every file, and so nothing in the journal is reachable only by
> luck. The orchestrator appends one line here every beat, in order. This is the index discipline the method
> teaches: a journal without an index is a pile, not a record.

Append-only. One line per beat, newest at the bottom. Never renumber, never rewrite a past line; if a beat is
superseded, the superseding beat gets its own line that says so.
