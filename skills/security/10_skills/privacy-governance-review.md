# Privacy Governance Review

## Required Checks

- Personal, confidential, privileged, regulated, and customer data are classified.
- Data minimization, retention, deletion, and export paths are explicit.
- Logs, analytics, support workflows, and AI/tool calls avoid unnecessary sensitive data.
- Access to sensitive data is auditable and role-scoped.
- Public documentation, security pages, and procurement packets match actual controls.

## Output

Record data classes, risky flows, evidence gaps, and any compliance or customer-facing claims that need correction.
