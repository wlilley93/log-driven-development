# Procurement Security Adapter

Apply when preparing customer-facing security packets, trust pages, SOC2 evidence, or vendor questionnaire responses.

## Required Checks

- Claims match implemented controls and current evidence.
- Any exceptions, roadmap items, or partial controls are not presented as complete.
- Sensitive architecture, customer, or incident details are redacted appropriately.
- Evidence points to reproducible commands, policies, tickets, or reports.
