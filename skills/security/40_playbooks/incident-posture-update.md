# Incident Posture Update

Use after a confirmed or suspected security defect.

Capture:

- Failure mode and affected surfaces.
- Exploitability, blast radius, and impacted data classes.
- Why existing skills, tests, scanners, or compliance scripts missed it.
- Code fix requirements.
- Behavioral regression tests.
- Security-Suite skill, adapter, methodology, or audit-script updates needed to catch the class next time.
- Documentation, evidence, customer/support/procurement updates.
- Follow-up monitoring and owner.
